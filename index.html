How to use the Counter:

  1. Start with a number

    • The app should start counting from 0 (or any default number).

  2. Increase button

    • When the user clicks Increase (+) → the number should increase by 1.

      • Example:
      0 → 1 → 2 → 3


  3. Decrease button

    • When the user clicks Decrease (-) → the number should decrease by 1.

      • Example:
      3 → 2 → 1 → 0


  4. Reset button

    • When the user clicks Reset → the number should go back to 0.
